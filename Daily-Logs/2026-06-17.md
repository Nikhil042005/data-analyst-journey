### ALTER TABLE Operations

* Adding a new column using `ADD COLUMN`
* Dropping an existing column using `DROP COLUMN`
* Modifying the data type of a column using `MODIFY`
* Changing the position/order of columns in a table

## Key Takeaways

* A database must be selected before creating tables.
* Choosing the correct data type is important for data integrity.
* The ALTER TABLE command is useful for modifying an existing table structure without recreating it.
